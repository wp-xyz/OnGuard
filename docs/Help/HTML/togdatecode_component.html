TOgDateCode Component

See also: properties, events, methods


Contents

 

TOgDateCode implements a Start/End Date release code. Use this release code when you need to limit the amount of time that an application (or specific features of an application) can be used. Both a start date and an end date are encoded into this release code. This allows you to detect a change to the computer’s clock that results in a date outside of the date range or an attempt to alter the registry or INI file entry.

OgProExeUtil.IsFileTampered

 
function IsFileTampered(const FileName : string; CheckSize : Boolean) : TExeStatus;


Contents

Use this function to check any file that has been stamped.

To apply a stamp to a file you must have the ogstamp.pas included within it as this includes the required record structure.

 

The default result is exeIntegrityError.

If the file has a stamp that is not yet used the result will be exeNotStamped.

If CheckSize is true the file's size is checked against the stamp's file size field, exeSizeError is returned if they do not match.

 

 

This function is only available on Windows.
